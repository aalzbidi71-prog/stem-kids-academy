منصة تعليمية عربية تفاعلية متكاملة مصممة خصيصاً وعلمياً لإلهام عقول الأطفال من سن 5 إلى 15 سنة وتأسيسهم في علوم المستقبل (الرياضيات الذهنية، اللغة الإنجليزية الحديثة، التفكير البرمجي الابتكاري، مبادئ الذكاء الاصطناعي).

A highly interactive Arabic-first educational academy engineered to inspire young minds (ages 5–15) in future-ready skills: Mental Mathematics, Modern English Phonetic mastery, Creative Programming, and AI fundamentals.

## 🌟 الميزات الفريدة للمشروع | High-Impact Features

### 1. 🤖 المستشار الذكي المتفاعل (StemMascotAI)
مساعد معزز بالذكاء الاصطناعي (Express API & Gemini 3.5 Flash) يتغير سلوكه ووضعية الزرافة وفقًا لنمط تفاعل الطفل:
* **محدد المسار الأكاديمي (Pathfinder):** استماع ودي ومراعٍ لمستويات الأطفال وتوجيههم للمسار الأمثل.
* **صانع التحديات الرياضية (Challenger):** ابتكار ألغاز ذكاء، ومسائل حساب ذهني مخصصة.
* **مبسط العلوم التقنية (Explainer):** رواية شرح مبسط بقصص خيالية لتقريب المفاهيم الصعبة (منطق، تمثيل حسي للأرقام).

### 2. 🎮 مختبر المشاريع والمهارات التفاعلي (Sandbox Classroom)
* **ركن الحساب الذهني التفاعلي (Abacus Arena):** تتبع تفاعلي واستجابات حية لحل ألغاز الحساب السريع.
* **ركن لفظ الإنجليزية (Pronunciation Master):** يتيح للأطفال التدرب السمعي والتدقيق اللفظي السليم بدقة عالية.
* **ركن التشفير الابتكاري (Block Playground):** بناء تدرجات منطقية تفاعلية لشرح علوم الكومبيوتر البرمجية بصرياً.

### 3. 💳 حاسبة الباقات وتسريع الحجز (Checkout flow Optimization)
* دعم الميزات التفاعلية لحساب فترات الاشتراك، الخصومات الحية الفورية للعائلات، وآلية دفع فوري بظهر زرافة التميز الحاملة للشهادة (`pose={6}`) لتعزيز شعور الثقة والتحفيز لدى الطفل وأسرته.

## 🚀 التشغيل والتطوير محلياً | Local Development Setup

تتبع المنصة بنية حديثة متكاملة (Full-stack Setup: Client inside Server Dev Proxy) لحماية وتأمين مفتاح الـ Gemini API في الخادم الخلفي دون تعريضها لمتصفح العميل:

This platform utilizes a secure Full-stack structure where the Gemini API stays entirely on the server-side proxy to protect your API keys.

### 1- تثبيت المتطلبات | Installation
تأكد من توفر الإصدار 18+ من Node.js، ثم قم بتثبيت الاعتمادات والحزم عبر موجه الأوامر:
```bash
npm install
```

### 2- إعداد مفاتيح البيئة والذكاء الاصطناعي | Configure Keys
قم بنسخ ملف المتغيرات التجريبي وتسميته باسم `.env`:
```bash
cp .env.example .env
```
افتح ملف `.env` وقم بوضع مفتاح الذكاء الاصطناعي الفريد الخاص بك:
```env
GEMINI_API_KEY="AIzaSyYourOwnGeminiKeyHere"
```

### 3- تفعيل بيئة التطوير المحلية | Standard Dev Server
لتشغيل المشروع محلياً متكاملاً مع الخادم الخلفي ومحرك التحديث البصري الفوري السريع:
```bash
npm run dev
```
تصفح بعدها عبر الرابط التالي مباشرة:
👉 **[http://localhost:3000](http://localhost:3000)**

### 4- بناء الحزمة للإنتاج | Build for Production
لتحزيم المشروع وإنتاج ملفات الإنتاج فائقة السرعة المترابطة داخل مجلد الـ `dist`:
```bash
npm run build
```

### 5- تشغيل خادم الإنتاج محلياً | Run Production Build
```bash
npm run start
```

## ☁️ خطوات النشر والاستضافة | Deployment Guide

المنصة مصممة وجاهزة للنشر الفوري على السحابة الإلكترونية (Render, Heroku, Cloud Run, Vercel) بمجرد رفع الكود إلى مستودع الجيت هاب الخاص بك:

This platform is production-ready for instantaneous deployment using:
1. **GitHub Flow**: Push this code to your repository.
2. **Environment variable setup**: Assign `GEMINI_API_KEY` on your hosting cloud dashboard (e.g., Render Env Settings).
3. **Build & Start commands**: 
   * **Build command:** `npm run build`
   * **Start command:** `npm run start`
